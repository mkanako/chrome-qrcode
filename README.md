# chrome扩展 - 二维码生成器

## Features
* 默认生成当前页面二维码，点击二维码进入编辑

## Install
1. [下载](https://github.com/mkanako/chrome-qrcode/archive/master.zip)仓库代码并解压
2. 打开`chrome://extensions/`
3. 启用`开发者模式`
4. 点击`加载已解压的扩展程序`
5. 选择上面的解压目录
