# chrome扩展-二维码生成器

![img](https://github.com/mkanako/chrome_extension_qrcode/blob/master/Screenshot.png?raw=true)

## Features
* 默认生成当前页面二维码，可在输入框修改生成其他字符串的二维码

* 二维码大小根据长度自动调整

* 右键链接或者选择文本右键也可生成二维码

##Dependences
[qrcodejs](https://github.com/davidshimjs/qrcodejs)

[vue2](https://github.com/vuejs/vue)
