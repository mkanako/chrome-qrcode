# chrome扩展 - 二维码生成器

![img](https://github.com/mkanako/chrome-qrcode/blob/master/Screenshot.png?raw=true)

## Features
* 默认生成当前页面二维码，点击二维码进入编辑

* 二维码大小根据文本长度自动调整

* 右键链接或者选择文本右键也可生成二维码

## Dependences
[qrcodejs](https://github.com/davidshimjs/qrcodejs)
